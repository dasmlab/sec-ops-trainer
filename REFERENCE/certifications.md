# Certifications Guide

## Overview

This guide outlines recommended certifications for each tier of the SecOps Standards Officer competency development path. Certifications validate knowledge and provide industry recognition.

## Certification Philosophy

- **Foundation Tier**: Focus on foundational certifications (RHCSA, CCNA)
- **Intermediate Tier**: Build specialized expertise (RHCE, Security certifications)
- **Advanced Tier**: Advanced certifications (CISSP, RHCA, Lead Auditor)
- **Expert Tier**: Maintain certifications, pursue advanced specializations

---

## Foundation Tier Certifications

### Required Certifications

#### RHCSA (Red Hat Certified System Administrator)
- **Provider**: Red Hat
- **Exam**: EX200
- **Prerequisites**: RH124 + RH134 or equivalent experience
- **Status**: **REQUIRED before Intermediate Tier**
- **Link**: [RHCSA Certification](https://www.redhat.com/en/services/certification/rhcsa)
- **Focus**: Linux/RHEL fundamentals, system administration

### Highly Recommended Certifications

#### CCNA (Cisco Certified Network Associate)
- **Provider**: Cisco
- **Prerequisites**: None (entry-level)
- **Status**: **HIGHLY RECOMMENDED**
- **Link**: [CCNA Certification](https://www.cisco.com/c/en/us/training-events/training-certifications/certifications/associate/ccna.html)
- **Focus**: Networking fundamentals, OSI model, routing/switching

### Recommended Certifications

#### CompTIA Security+
- **Provider**: CompTIA
- **Prerequisites**: None (entry-level)
- **Status**: **RECOMMENDED**
- **Link**: [CompTIA Security+](https://www.comptia.org/certifications/security)
- **Focus**: Security fundamentals, broad security knowledge

#### CompTIA Network+
- **Provider**: CompTIA
- **Prerequisites**: None (entry-level)
- **Status**: **RECOMMENDED** (if not pursuing CCNA)
- **Link**: [CompTIA Network+](https://www.comptia.org/certifications/network)
- **Focus**: Networking fundamentals (vendor-neutral)

## Intermediate Tier Certifications

### Required Certifications

#### RHCE (Red Hat Certified Engineer)
- **Provider**: Red Hat
- **Exam**: EX294 (Ansible automation)
- **Prerequisites**: RHCSA
- **Status**: **HIGHLY RECOMMENDED**
- **Link**: [RHCE Certification](https://www.redhat.com/en/services/certification/rhce)
- **Focus**: Advanced Linux administration, automation

### Recommended Certifications

#### RHCSE (Red Hat Certified Specialist in Security: Linux)
- **Provider**: Red Hat
- **Prerequisites**: RHCSA (recommended)
- **Status**: **RECOMMENDED**
- **Link**: [RHCSE Certification](https://www.redhat.com/en/services/certification/red-hat-certified-specialist-security-linux)
- **Focus**: Linux security specialization

#### CCNP Enterprise
- **Provider**: Cisco
- **Prerequisites**: CCNA
- **Status**: **RECOMMENDED** (if networking focus)
- **Link**: [CCNP Enterprise](https://www.cisco.com/c/en/us/training-events/training-certifications/certifications/professional/ccnp-enterprise.html)
- **Focus**: Advanced routing and switching

#### CCNP Security
- **Provider**: Cisco
- **Prerequisites**: CCNA
- **Status**: **RECOMMENDED** (if security focus)
- **Link**: [CCNP Security](https://www.cisco.com/c/en/us/training-events/training-certifications/certifications/professional/ccnp-security.html)
- **Focus**: Network security, firewall, VPN

#### CISSP (Certified Information Systems Security Professional)
- **Provider**: ISC²
- **Prerequisites**: 5 years security experience (or 4 years + degree/cert)
- **Status**: **HIGHLY RECOMMENDED** (can pursue in Intermediate or Advanced Tier)
- **Link**: [CISSP Certification](https://www.isc2.org/cissp)
- **Focus**: Comprehensive security knowledge, governance, risk management

#### GIAC Certifications (SANS)
- **Provider**: SANS / GIAC
- **Prerequisites**: SANS training courses
- **Status**: **RECOMMENDED** (various specializations)
- **Link**: [GIAC Certifications](https://www.giac.org/certifications)
- **Examples**: 
  - GSEC (Security Essentials)
  - GCIH (Incident Handler)
  - GPEN (Penetration Tester)

## Advanced Tier Certifications

### Highly Recommended Certifications

#### CISSP (Certified Information Systems Security Professional)
- **Provider**: ISC²
- **Prerequisites**: 5 years security experience (or 4 years + degree/cert)
- **Status**: **HIGHLY RECOMMENDED for Advanced Tier**
- **Link**: [CISSP Certification](https://www.isc2.org/cissp)
- **Focus**: Security governance, risk management, policy development

#### RHCA (Red Hat Certified Architect)
- **Provider**: Red Hat
- **Prerequisites**: Requires 5 Red Hat certifications
- **Status**: **RECOMMENDED for Advanced Tier**
- **Link**: [RHCA Certification](https://www.redhat.com/en/services/certification/rhca)
- **Focus**: Advanced architecture, specialization tracks
- **Tracks**: Security, Cloud, DevOps, Infrastructure

#### ISO 27001 Lead Auditor / Lead Implementer
- **Provider**: Various (BSI, DNV, etc.)
- **Prerequisites**: ISO 27001 training
- **Status**: **RECOMMENDED** (if ISO focus)
- **Focus**: ISO 27001 ISMS implementation/audit

### Recommended Certifications

#### Advanced ISC² Certifications
- **CCSP (Certified Cloud Security Professional)**
  - [CCSP Certification](https://www.isc2.org/ccsp)
  - Cloud security focus
- **CSSLP (Certified Secure Software Lifecycle Professional)**
  - [CSSLP Certification](https://www.isc2.org/csslp)
  - Application security focus

#### Advanced GIAC Certifications (SANS)
- **GSTRT (Strategic Planning, Policy, and Leadership)**
  - Strategic security leadership
- **GCPM (Continuous Monitoring)**
  - Security monitoring and operations
- **Various specializations** based on career focus

#### Cloud Security Certifications
- **AWS Certified Security - Specialty**
  - [AWS Security Certification](https://aws.amazon.com/certification/certified-security-specialty/)
- **Azure Security Engineer Associate**
  - [Azure Security Certification](https://docs.microsoft.com/en-us/learn/certifications/azure-security-engineer)
- **Google Professional Cloud Security Engineer**
  - [Google Cloud Security Certification](https://cloud.google.com/certification/cloud-security-engineer)

#### Container/Kubernetes Security
- **CKS (Certified Kubernetes Security Specialist)**
  - [CKS Certification](https://www.cncf.io/certification/cks/)
  - Kubernetes security focus
- **CKA (Certified Kubernetes Administrator)**
  - [CKA Certification](https://www.cncf.io/certification/cka/)
  - Kubernetes administration

#### Red Hat Specializations
- **Red Hat Certified Specialist in OpenShift Administration**
  - [OpenShift Certification](https://www.redhat.com/en/services/certification/openshift)
- **Red Hat Certified Specialist in Ansible Automation**
  - [Ansible Certification](https://www.redhat.com/en/services/certification/ansible)
- **Red Hat Certified Specialist in Identity Management**
  - [Identity Management Certification](https://www.redhat.com/en/services/certification/identity-management)

## Expert Tier Certifications

### Maintained Certifications

At Expert Tier, maintain existing certifications:
- CISSP (with CPE credits)
- RHCA (with recertification)
- Industry certifications (with renewal)
- Specialized certifications (as relevant)

### Advanced Specializations

Pursue advanced certifications based on specialization:
- **Executive security programs**
- **Board-level security certifications**
- **Industry-specific certifications**
- **Standards body certifications**
- **Advanced vendor certifications**

---

## Certification Maintenance

### Continuing Education Requirements

Most certifications require continuing education:
- **CISSP**: 40 CPE credits per year (120 over 3 years)
- **RHCE/RHCA**: Maintain through additional certifications or training
- **GIAC**: Maintain through continuing education
- **Vendor certifications**: Vary by vendor

### Recertification
- Track recertification requirements
- Plan continuing education activities
- Maintain certification currency
- Document professional development

## Certification Path Recommendations

### RHEL-Focused Path
1. **Foundation**: RHCSA (REQUIRED)
2. **Intermediate**: RHCE, RHCSE (RECOMMENDED)
3. **Advanced**: RHCA (RECOMMENDED)
4. **Expert**: Maintain RHCA, pursue specializations

### Networking-Focused Path
1. **Foundation**: CCNA (HIGHLY RECOMMENDED)
2. **Intermediate**: CCNP Enterprise, CCNP Security (RECOMMENDED)
3. **Advanced**: CCIE (Optional, if networking specialist)
4. **Expert**: Maintain certifications

### Security-Focused Path
1. **Foundation**: Security+ (RECOMMENDED)
2. **Intermediate**: CISSP (HIGHLY RECOMMENDED)
3. **Advanced**: Advanced ISC², GIAC specializations
4. **Expert**: Maintain certifications, pursue leadership certifications

### Standards/Compliance-Focused Path
1. **Foundation**: Security+ (RECOMMENDED)
2. **Intermediate**: CISSP (HIGHLY RECOMMENDED)
3. **Advanced**: ISO 27001 Lead Auditor/Implementer, specialized certifications
4. **Expert**: Maintain certifications, standards body certifications

### Cloud-Focused Path
1. **Foundation**: RHCSA, CCNA (RECOMMENDED)
2. **Intermediate**: RHCE, Cloud fundamentals (AWS/Azure/GCP)
3. **Advanced**: CCSP, Cloud Security Specialty certifications
4. **Expert**: Maintain cloud certifications, advanced cloud specializations

## Certification Planning Tips

1. **Plan Early**: Identify target certifications for each tier
2. **Prerequisites**: Understand certification prerequisites
3. **Training**: Align training with certification goals
4. **Practice**: Use labs and practice exams
5. **Schedule**: Plan exam dates and study time
6. **Maintain**: Track continuing education requirements
7. **Network**: Connect with certified professionals
8. **Document**: Maintain certification records
